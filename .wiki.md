# Project Summary
MGX Backend is a simplified implementation of the MetaGPT multi-agent software development system. It leverages OpenAI's API to facilitate collaborative project generation by simulating the roles of a Product Manager, Architect, and Engineer, enabling users to create fully functional software projects from high-level ideas.

# Project Module Description
The project consists of several functional modules:
- **Roles**: Define the roles (ProductManager, Architect, Engineer) that interact and perform tasks.
- **Actions**: Implement actions (WritePRD, WriteDesign, WriteCode) that each role can execute.
- **Team Management**: Manages the collaboration between roles and oversees project execution.
- **Project Repository**: Organizes and saves the generated project files.
- **Cost Management**: Tracks API usage and ensures budget limits are adhered to.

# Usage
### Installation
```bash
cd mgx_backend
pip install -r requirements.txt
```

### Set API Key
```bash
export OPENAI_API_KEY="your-api-key-here"
```

### Quick Start
```python
from mgx_backend.software_company import generate_repo

# Generate a project
project_path = generate_repo("Create a 2048 game")
print(f"Project created at: {project_path}")
```

### CLI Usage
```bash
python cli.py "Create a calculator app" --investment 5.0
```

### Advanced Usage
```python
from mgx_backend.config import Config
from mgx_backend.context import Context
from mgx_backend.team import Team
from mgx_backend.roles import ProductManager, Architect, Engineer

async def create_custom_project():
    config = Config.default()
    ctx = Context(config=config)
    team = Team(context=ctx)
    team.hire([ProductManager(), Architect(), Engineer()])
    team.invest(10.0)
    await team.run(n_round=5, idea="Create a scientific calculator web application")
```
